# Cub3D — Raycasting Engine

A 3D graphical project inspired by Wolfenstein 3D, built using C and the MinilibX library.

## 🕹️ Controls
- **W, A, S, D:** Move the player.
- **Left / Right Arrows:** Rotate the camera.
- **ESC / Close Window:** Exit the game.
