# FdF — 3D Wireframe Renderer

A program that reads a coordinates map and renders a 3D wireframe representation using isometric projection.

## ⌨️ Controls
- **Arrows / WASD:** Move the map.
- **+/- or Scroll:** Zoom in/out.
- **PageUp / PageDown:** Change altitude (Z-axis).
- **R:** Reset view.

## 🚀 Usage
```bash
make
./fdf maps/42.fdf
